# UI/UX Variants

Three design variants for the chat app. Each folder contains complete replacements for the main style files.

## How to apply a variant

Copy the files from your chosen variant folder to replace the originals:

| Variant file | Replace |
|---|---|
| `index.css` | `src/index.css` |
| `Homepage.module.scss` | `src/components/Homepage.module.scss` |
| `ChatCard.module.scss` | `src/components/chatCard/ChatCard.module.scss` |
| `MesaggePage.module.scss` | `src/components/messagePage/MesaggePage.module.scss` |
| `MessageCard.module.scss` | `src/components/messageCard/MessageCard.module.scss` |
| `Register.module.scss` | `src/components/register/Register.module.scss` |

---

## Variant 1 — Dark Midnight
Full dark theme. Dark backgrounds, bright green accent, gold highlights.
Folder: `variant1-dark/`

## Variant 2 — Clean Minimal
Pure white, no gradients in the header, very clean and airy.
Folder: `variant2-minimal/`

## Variant 3 — Glassmorphism
Frosted glass on a deep gradient background. Modern, immersive look.
Folder: `variant3-glass/`
